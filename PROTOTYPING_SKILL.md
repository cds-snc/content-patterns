# Content Prototyping Skill

This skill provides expertise in converting content designs into low-fidelity text-based prototypes using the organization's IA standards.

## IA Syntax
Use the follow syntax for interface components:
- **Buttons**: `[Button Label]`
- **Text Fields**: 
  ```
  Field Label
  [        ]
  ```
- **Checkboxes**: `[ ] Option`
- **Radio Buttons**: `( ) Option`
- **Headings**: `# Heading Level`
- **Breadcrumbs**: `Path > Sub-path`

## Application
When this skill is invoked, the agent should:
1. Review the content drafted according to [Design_principles.md](Design_principles.md).
2. Map the content to the IA components defined in [IA_prototyping.md](IA_prototyping.md).
3. Produce a structured text-based wireframe.
