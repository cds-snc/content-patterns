
## Goal
Things senders must know before sending a file:

- They need to upload the file through the UI, may be intuitive given there is a button
- When they send the file, the recipients will get the file as is, they can keep the copy in their inbox indefinitely. 
- Senders can't revoke or update the file. The recipient now owns it
- supported file types: .pdf, .doc, .docx, .odt, .rtf, txt, .xlsx, .csv, .json, .jpeg, .jpg, .png
- You can attach up to 10 files. 
- All files must not exceed 6 MB.
- Files will be scanned for malware, and then attached. You cannot  send files while they are being scanned.
